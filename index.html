Как настроить GitHub Pages

  • На странице вашего репозитория нажмите на вкладку Settings (Настройки).
  • В меню слева выберите раздел Pages (GitHub Pages).
  • В разделе Branch (Ветка) выберите ветку, из которой будет развертываться сайт (например, main).
  • Выберите каталог. Если файлы в корневом каталоге, выберите / (root).
  • Нажмите Save (Сохранить).
  • После сохранения GitHub укажет вам URL вашего сайта, например: https://username.github.io/repository-name/.
